# Sales Forecasting Project

This project predicts the next 8 weeks of sales using historical sales data.

Models Used:
- SARIMA
- Prophet
- XGBoost
- LSTM

Features:
- Data preprocessing
- Feature engineering
- Model comparison
- Forecast visualization
- FastAPI deployment structure

Tools Used:
- Python
- Google Colab
- Pandas
- Scikit-learn
- TensorFlow
- FastAPI
